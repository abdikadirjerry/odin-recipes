# odin-recipes# Odin Recipes

This is my first HTML project from The Odin Project.

I made a simple recipe website using only HTML. The website has a main page with links to different recipes. Each recipe has a description, ingredients, steps, and an image.

## Recipes

- Barbecue Chicken
- Cheeseburger
- Spaghetti Bolognese
- Margherita Pizza

## What I practiced

- Creating HTML pages
- Using headings and paragraphs
- Creating lists
- Adding images
- Creating links between pages

I know it's a simple project, but it helped me understand the basics of HTML.

Thanks for checking out my project!
